Exam Procedure

You can use papers with slides and other material.
You CANNOT use USB memories, CD-ROM, floppies, etc.
You CANNOT go out from the lab before the end of the exam, even if you don't want to submit your exam.

Once you get the zip file with the Eclipse project and the text of the exam.
  • Install Eclipse and Java
  • Open Eclipse and insert student-id ("matricola") and a valid e-mail address.  
  • Import the project following these instructions:
    • Install Power Archive program to unzip the file.
    • Download the zip file from the address at the end of this page.
    • Extract the zip file in a folder in a local disk (not on the desktop)
    • In Eclipse do these steps: File; Import; Existing project into workspace; 
    • Then, select the folder where you have unzipped the downloaded files.
  • Do the exam without modifying class names and methods signatures.
  • Before submitting your exam export the project (right-click on project name, export as zip, in your personal network disk P:\studentid-surname-name.zip)
  • Finally submit your project (right-click on the project and click "Sottometti progetto")
    • Note Well: the submitted project MUST be syntactically correct , that is: it MUST be compiled.

Download the zip with the initial project from : http://softeng.polito.it/01JEY/Exam2004-Trains.zip
